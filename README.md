
# Project Journal Web App

A minimal project journal web app built with Next.js (App Router), TypeScript, and Tailwind CSS.

## Features
- Home page: List of projects (name, last updated date)
- Project page: Timeline of daily entries (date, hours spent, notes)
- Add entry form: Hours spent, notes
- State stored in localStorage (no backend)
- Clean, minimal UI

## Getting Started

1. Install dependencies (if not already):
	```sh
	npm install
	```
2. Run the development server:
	```sh
	npm run dev
	```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack
- [Next.js](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Notes
- No authentication, backend, or database yet.
- All data is stored in localStorage.
